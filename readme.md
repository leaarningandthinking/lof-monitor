# LOF-Monitor | 基金套利实时监控助手

这是一个基于微信小程序开发的金融工具，旨在帮助投资者实时监控 LOF（上市开放式基金）和 QDII 基金的折溢价情况，快速锁定套利机会。

## 🌟 核心功能
* **实时监控**：同步监控 LOF 和 QDII 基金的现价与估值。
* **智能排序**：支持按溢价率进行降序、升序排列，一眼发现高溢价机会。
* **多维数据**：展示基金代码、名称、现价、估值及实时溢价率。

## 🛠️ 技术栈
* **前端**：微信小程序原生开发（WXML/WXSS/JS）。
* **后端**：部署于阿里云轻量应用服务器的 API 接口。
* **数据同步**：运行于本地（MacBook Air M1）的 Python 自动化抓取脚本。

## 🚀 快速开始
1. 克隆代码到本地。
2. 使用微信开发者工具打开项目。
3. 在 `index.js` 中配置你的服务器 API 地址。
4. 开启开发者工具的“不校验合法域名”选项。
